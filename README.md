# \# AI-Based Behavioral Intrusion Detection System (IDS)

# 

# \## Project Overview

# This project is a web-based Intrusion Detection System that utilizes an Unsupervised Machine Learning model (Isolation Forest) to detect anomalous and potentially malicious network behavior. It includes a backend API powered by Flask and a frontend analytics dashboard.
